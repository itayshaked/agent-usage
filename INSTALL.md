# Install Cursor Usage (menu bar app)

A tiny macOS menu bar app that shows your Cursor usage and how close you are to
your limit. It reads your **already signed-in Cursor app** — nothing to paste.

Requirements: macOS 13+, and the **Cursor app installed and signed in**.

## Install

The app is signed with a Developer ID and notarized by Apple, so it opens
normally — no warnings, no workarounds.

1. Download `CursorUsageBar.zip` and unzip it.
2. Drag **CursorUsageBar** into your **Applications** folder.
3. Double-click to open.

Or via Homebrew (see the main README) for a one-line install + easy updates.

## Tips

- Click the menu bar icon to see your included-usage bar and per-model spend.
- Gear menu → **Launch at login** to keep it running automatically.
- The icon turns **orange** past 70% and **red** past 90% of your limit.
